## First Step

You have to create a project repository in Github. Then, you connect
them to RStudio by using usethis::create_from_github(#paste the link
here HTTPS) or you can also do it by File \> New Project \> Version
Control \> Git. In the “repository URL” paste the URL of your new GitHub
repository.
